Mapping File Format

The file should contain the following columns:

  • source_table: Source table name
  • target_table: Target table name
  • source_columns (optional): Source table columns
  • target_columns (optional): Target table columns
  • transformations (optional): Transformation rules
  • where_condition (optional): WHERE condition to filter data
